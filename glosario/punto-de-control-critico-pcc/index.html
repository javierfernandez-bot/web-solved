Punto de Control Crítico (PCC)

¿Qué es un Punto de Control Crítico (PCC)?

Un Punto de Control Crítico (PCC) es la fase de un proceso alimentario en la que se aplica un control esencial para prevenir, eliminar o reducir a un nivel aceptable un peligro para la seguridad del alimento. Es un concepto central del sistema APPCC.

Cada PCC exige límites críticos definidos, vigilancia continua, acciones correctivas y registros documentados.
