Control estadístico de proceso (SPC)

¿Qué es el control estadístico de proceso (SPC)?

El control estadístico de proceso (SPC) es el uso de datos y gráficos de control para vigilar la variabilidad de un proceso productivo y detectar desviaciones antes de que generen producto no conforme. Distingue la variación normal de la que requiere intervención.

Aumenta la estabilidad del proceso, reduce mermas y aporta base objetiva para la mejora continua.
